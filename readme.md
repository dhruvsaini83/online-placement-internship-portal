# Online Placement & Internship Portal

A web-based platform to manage college placement and internship activities.

## Features

- Student registration & login
- Company login & job posting
- Job / Internship listing
- Apply system
- Application tracking
- Admin dashboard

## Tech Stack

- Node.js
- Express.js
- MongoDB
- EJS
- Passport.js

## Project Type

MCA IV Semester Industrial Major Project
